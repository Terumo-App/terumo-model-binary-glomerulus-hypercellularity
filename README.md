# Terumo: binary classifiers for glomerular data

- Usage:

1. Copy all the dataset to the `data/raw` folder.
2. Arrange the dataset into 6 sub-folders of the `data/raw` folder:
    1. `Hypercellularity`
    2. `Crescent`
    3. `Normal`
    4. `Podocytopathy`
    5. `Membranous`
    6. `Sclerosis`
3. Run `make build_dataset`
